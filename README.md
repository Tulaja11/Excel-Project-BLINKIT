# Blinkit Grocery Sales Analysis (Excel Dashboard)

## Project Overview
- This project is an Excel-based interactive dashboard built using a Blinkit grocery dataset.
- The goal is to analyze sales performance, customer ratings, outlet trends, and product insights to support data-driven business decisions.
- The dashboard transforms raw grocery data into meaningful insights using Excelâ€™s data analysis and visualization features.

## Objectives
- Analyze total and average sales performance
- Compare sales across outlet sizes, tiers, and locations
- Understand product category performance
- Evaluate fat content impact on sales
- Track outlet establishment trends over time

## Dashboard Features
KPI Cards
- Total Sales
- Average Sales
- Number of Items
- Average Rating

Visual Insights
- Sales by Outlet Establishment Year
- Sales by Outlet Size
- Sales by Outlet Location
- Sales by Item Type
- Fat Content Analysis
- Outlet Type Comparison

Interactive Filters (Slicers)
- Outlet Size
- Outlet Location Tier
- Item Type

## Tools & Skills Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers & Filters
- Data Cleaning
- Data Visualization

## Key Insights
- Tier 3 outlets generated strong sales contribution
- Regular fat products had higher sales share
- Supermarket Type 1 outlets dominated total sales
- Certain item categories like Fruits, Snacks, and Household products performed best
